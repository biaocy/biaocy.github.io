腾讯云实例docker配置

发表于 2023-04-18 更新于 2023-04-19

本文是关于docker安装源的配置，不是镜像源的配置
镜像源配置请参考这里

原因

出于小程序后端需要，在腾讯云购买了一台轻量服务器，系统是 Debian 11 (bullseye)，可用区域在广州。
完成基本的系统配置后，开始按照docker 官网教程[1] 安装docker，发现安装过程非常慢。

解决方案

这个问题可以通过修改 docker 安装源地址为腾讯云的 docker 安装源地址来解决。
因为这个实例是 Debian 11 (bullseye)，所以对应安装源地址是 https://mirrors.cloud.tencent.com/docker-ce/linux/debian/

注意，上述keyring docker.gpg 用 docker 官网的即可，因为源地址的binary都是copy过来的。

  1. https://docs.docker.com/engine/install/debian/#install-using-the-repository ↩︎

  1. 非Windows系统
